LISTS In HTML

List:-
In HTML, a list is a way to group related items together

Types Of List :-
  1. Ordered List :-
    An ordered list is a collection of items arranged in a specific sequence. The browser automatically numbers each item.

    Syntax:-

    < ol type="1" >
    < li > Rohan < /li >
    < li > SUMIT < /li >
    < li > CHANDRA < /li >
    < /ol >

    OUTPUT:-
    1. Rohan
    2. SUMIT
    3. CHANDRA
    Types :-
    1. Number : — type="1"
      • Output:
        1. Item 1
        2. Item 2
        3. Item 3
    2. Uppercase Letters : — type="A"
      • Output:
        1. Item 1
        2. Item 2
        3. Item 3
    3. Lowercase Letters : — type="a"
      • Output:
        1. Item 1
        2. Item 2
        3. Item 3
    4. Uppercase Roman Numerals : — type="I"
      • Output:
        1. Item 1
        2. Item 2
        3. Item 3
    5. Lowercase Roman Numerals : — type="i"
      • Output:
        1. Item 1
        2. Item 2
        3. Item 3
  2. Unordered List :-
    3. An unordered list is a collection of items where the order doesn’t matter. Items are usually displayed with bullets.


    Syntax:-

    < ul type="" >
    < li > Rohan < /li >
    < li > SUMIT < /li >
    < li > CHANDRA < /li >
    < /ul >

    OUTPUT:-
    • Rohan
    • SUMIT
    • CHANDRA
    Types :-
    1. Disc (default) : — type="disc"
      • Output:
        • Item 1
        • Item 2
        • Item 3
    2. Circle : — type="Circle"
      • Output:
        • Item 1
        • Item 2
        • Item 3
    3. Square : — type="Square"
      • Output:
        • Item 1
        • Item 2
        • Item 3
  3. Description List:-
    4. A description list is used to define terms and their descriptions. It uses <dt> for the term and <dd> for the description.

    Example:-


    <dt>
    <dd>HTML<dd>
    <dd>A language for creating web pages.<dd>
    <dt>CSS </dt>
    <dd>Used to style web pages./</dd>
    </dt>



    Output:-

    HTML
    A language for creating web pages.
    CSS
    Used to style web pages.
