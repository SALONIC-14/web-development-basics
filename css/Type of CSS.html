1. Inline Css
    2.
    CSS is written directly inside the HTML element using the style attribute.
    • Syntax :- <h1 style="color: red; font-size: 30px;">Hello World</h1>
    • Output:-

      Hello World

  2. Internal Css
    3.
    • CSS is written inside the <style> tag in the <head> section of the HTML file.
    • Styles apply only to that single page.
    • Syntax:- 
                              <!DOCTYPE html>
                        <html>
                        <head>
                        <style>
                            h1 {
                            color: blue;
                            text-align: center;
                            }
                        </style>
                        </head>
                        <body>
                        <h1>Hello World </h1>
                        < /body >
                        < /html >
    • Output:-

      • Hello World

  3. External Css>
    • CSS is written in a separate .css file and linked to the HTML using <link>.
    • Best for large websites because the same stylesheet can be reused.
    • Syntax:- 
                          <!DOCTYPE html>
                    <html>
                    <head>
                    <link rel="stylesheet" type="text/css" href="style.css">
                    </head>
                    <body>
                    <h1>Hello World</h1>
                    </body>
                    </html>

    • Output :-

      Hello World
